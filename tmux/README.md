## Shortcuts

| Raccourcis                 | Actions                  |
| --------------------------:| ------------------------:|
| ```Ctrl``` + ```Space```   | Prefix Mode              |
|              ```d```       | Detach                   |
|              ```Space```   | choose-session           |
|              ```a```       | Reload Config            |
|              ```Enter```   | Split (current tree)     |
|              ```r```       | Rotate the tree          |
|              ```Up```      | Split window verticaly   |
|              ```Right```   | Split wimdow horizontaly |
|              ```s```       | Sync pane                |
|              ```q```       | Close the current window |
|              ```w```       | Close the current pane   |
|              ```Tab```     | Next Window              |
